Set up the AI model

Ash runs entirely on your machine — no account, no cost, fully offline. The engine and index ship with the app; only the models download, once.

Models & runtime
  • llama.cpp↗
    Runs the models on your machine's GPU.
  • llama-swap↗
    Serves all three models behind one local endpoint.
  • khora↗
    The on-device index and graph — stores descriptions and links your photos. SQLite + LanceDB, no server.
  • SmolVLM2 2.2B↗
    Vision model — looks at each photo and writes the description.
  • Nomic Embed↗
    Embeddings — turns descriptions into vectors so you can search by meaning.
  • Qwen2.5 3B↗
    Text model — pulls out places, objects, and scenes to link related photos.
Rescan library

Re-describe re-runs the vision model on every photo. Re-extract only rebuilds the connections from existing descriptions — faster, good after a model change.

Immich

Pull photos from a self-hosted Immich server into your local library. Originals are downloaded and kept on this machine, then described and searchable like any other photo.
